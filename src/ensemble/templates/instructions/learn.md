# Learn Instructions

## 自己改善フロー

1. **直近のタスク実行を分析**:
   - completion-summary.yaml を確認
   - 成功/失敗の理由を分析
   - ボトルネックを特定

2. **改善提案を作成**:
   - MEMORY.md更新内容を提案
   - 具体的な改善点を列挙:
     - 新しく学んだパターン
     - 避けるべき失敗パターン
     - 効率化の提案

3. **提案をConductorに報告**:
   - `queue/learning/${SESSION_ID}.yaml` を作成
   - Conductorの承認を待つ

4. **承認後にMEMORY.md更新**:
   - 承認された提案のみを反映
   - 既存内容と重複しないよう注意
